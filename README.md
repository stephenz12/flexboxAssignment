# Flexbox Layout Assignment

## Description
This project demonstrates the use of **CSS Flexbox** to create responsive layouts.  
It shows how flex containers and flex items work together to align and distribute elements efficiently.

## Features
- Uses `display: flex` to create flexible layouts
- Demonstrates `justify-content`, `align-items`, and `flex-direction`
- Responsive design that adjusts to different screen sizes
- Includes multiple flex containers with various arrangements

## How to Use
1. Open `index.html` in your browser.
2. Resize the browser window to see the flex layout adapt.
3. Inspect the CSS to see how different flex properties affect the layout.

## Technologies Used
- HTML
- CSS (Flexbox)
